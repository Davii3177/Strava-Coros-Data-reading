Connection setup instructions

OAuth tokens stay on the server. Add credentials to your host environment and never paste refresh tokens into the browser.

  • Strava: run python src/strava_auth_server.py, finish authorization, then save STRAVA_REFRESH_TOKEN.
  • Google Health / Fitbit: run python src/fitbit_auth_server.py, sign in, then save GOOGLE_HEALTH_REFRESH_TOKEN.
  • COROS: hosted OAuth is not supported yet; it remains unavailable until a provider integration is configured.
