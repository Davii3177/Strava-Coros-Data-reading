Secure server setup

Connect or reconnect a source

Gaman keeps OAuth tokens on the server. Add the matching credentials to your host’s environment variables, then use the local authorization helper to obtain a refresh token. Never paste a refresh token into the browser.

  • Strava: run python src/strava_auth_server.py, complete the OAuth consent screen, then store the returned token as STRAVA_REFRESH_TOKEN.
  • Google Health / Fitbit: run python src/fitbit_auth_server.py, sign in with the Google account linked to Fitbit, then store GOOGLE_HEALTH_REFRESH_TOKEN. Reconnect if the status is expired or error.
  • COROS: its client is scaffolded but does not yet have a supported hosted OAuth flow, so it remains unavailable until a provider integration is configured.
